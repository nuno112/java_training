Contact List Exercise:



Create a Contact List app with the following requirements:

  • Contacts Name;
  • Contacts Phone Number;
  • Contacts Email;
  • Other contact options (at your choice);

Search Bar with the following filters:

  • Search by name;
  • Search by phone number;
  • Search by Email;
  • Other filters options (at your choice);

There must be only one input type (add/search contact) and the search filter must be choosen with a button or a dropdown list at your choice.

If no filter has been choosen, the search must look in all fields.

The contacts must be automatically ordered by Name.

Each contact must have a checkbox and there must be a button to delete all checked contacts.
